# AIPaperDraftingPrompt
—— 逐字逐句写论文：AI 学术会议论文起草 Prompt

> **项目初衷**：让每一位 CS 研究生都能拥有一位“AI 博导”作为写作搭档。
>
> **免责声明**：本项目提供的 Prompt 仅作为一种**论文写作逻辑、结构规范和语言风格的参考工具**。它旨在辅助思考与润色，绝非替代科研创新本身。请根据自身研究特点灵活调整，切勿生搬硬套。学术诚信是底线，AI 仅为助手。

## 📖 关于本项目

### 我的故事

我是一名刚毕业的CS博士。回首读博生涯，我最幸运的事莫过于遇到了一位极其负责的指导老师。在那些为了ACL、NeurIPS等顶会投稿而熬夜的日子里，老师曾逐字逐句地帮我修改每一篇会议论文。从 Introduction 到Method 的逻辑与表达，再到 Related Work 的结构划分与客观评价，这些宝贵的指导不仅帮助我发表了论文，更重塑了我的学术写作思维。

然而，我也深知并非每一位研究生都有幸遇到这样手把手指导的导师。许多同学在科研上颇有建树，却因写作逻辑不清、表达不地道而在审稿中遗憾被拒。

**因此，我决定将这套凝聚了导师心血与我个人实战经验的“论文写作提示词”开源。** 希望它能像一位耐心的“AI 博导”，在你迷茫时提供结构指引，在你词穷时提供风格参考。

### 这个项目是什么？

这是一套专为 **CS/AI 领域研究生** 设计的提示词集合，针对论文的大纲梳理搭建，帮大家的写作起个逻辑清晰的好头。

## 🚀 快速开始

### 适用场景

- 准备投稿 ICLR, ICML, NeurIPS, CVPR, ACL 等AI会议
- 英文学术写作基础薄弱，缺乏导师细致指导
- 需要对初稿进行写作逻辑和语言方面的检查与润色

### 使用建议

1. **分步执行**：不要试图一次性生成全文。按照：`结构拆解 -> Introduction撰写 -> 相关工作撰写 -> Abstract撰写 -> Method撰写（尚未包含）-> 精细打磨` 的顺序进行。
2. **提供参考样例**：Prompt 中预留了 `==xxx==` 占位符。**强烈建议**填入你目标领域内的高质量论文片段作为参考样例。
3. **人机协作**：AI 生成的内容必须经过人工核实。特别是参考文献、实验数据和核心创新点，务必确保真实性。
